TouchScreen.js Demo & Documentation

What is TouchScreen.js?

TouchScreen.js is a specialized JavaScript library that controls touch behavior and zoom functionality in web applications. It's particularly useful for kiosk-mode applications and touch-screen interfaces where you need to restrict certain touch behaviors.

Key Features

  • Multi-touch Prevention - Blocks gestures like pinch-to-zoom
  • Scale Control - Disables user scaling
  • Fullscreen Support - Adds a fullscreen button
  • Touch Parameter Propagation - Automatically adds touch parameters to links
  • Text Selection Prevention - Prevents text selection on touch devices

Implementation

1. Include the script in your HTML:

<script src="touchscreen.js"></script>

Activation

The script is activated by adding the parameter touchversion=true to your URL:

https://yourwebsite.com/page.html?touchversion=true

How It Works

  • When activated with touchversion=true, the script automatically:
  • Prevents multi-touch gestures like pinching
  • Adds meta tags to disable user scaling
  • Shows a fullscreen button in the top-right corner
  • Prevents context menus and text selection
  • Propagates the touch parameter to all HTML links automatically
